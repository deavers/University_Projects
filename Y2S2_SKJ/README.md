# Y2S2_SKJ – Scripting Languages (Python/Bash)

**Subject:** Scripting Languages  
**Semester:** Year 2, Semester 2 (Spring 2026)  
**Technology:** Python, Bash

---

## Course Goals

- Write robust automation and utility scripts in Python and Bash.
- Apply scripting to cyber security tasks (log analysis, file parsing, network tools).
- Understand shell scripting fundamentals: variables, loops, conditionals, and pipelines.
- Use Python libraries for text processing, networking, and OS interaction.
