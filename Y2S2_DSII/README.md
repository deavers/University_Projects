# Y2S2_DSII – Database Systems II (Django/Python)

**Subject:** Database Systems II  
**Semester:** Year 2, Semester 2 (Spring 2026)

## Course Goals

- Design and manage relational databases with advanced SQL features.
- Build full-stack web applications using the Django framework.
- Implement ORM-based data models, migrations, and queries.
- Understand database security, optimization, and indexing strategies.
