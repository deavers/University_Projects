# Y2S2_CPP – Advanced C++ Programming

**Subject:** Advanced C++ Programming  
**Semester:** Year 2, Semester 2 (Spring 2026)  
**Technology:** C++17/20

---

## Course Goals

- Master advanced C++ features: templates, lambdas, move semantics, RAII.
- Apply STL algorithms, containers, and iterators effectively.
- Understand memory management, smart pointers, and ownership models.
- Explore concurrency primitives: threads, mutexes, and futures.
- Write clean, idiomatic, and performant modern C++ code.
